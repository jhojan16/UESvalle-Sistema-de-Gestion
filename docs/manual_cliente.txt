MANUAL DE USUARIO Y OPERACION
UES VALLE - SISTEMA DE GESTION
Version de entrega: 2026-02-23

====================================================================
1) INTRODUCCION
====================================================================

Este documento describe como usar la aplicacion web UES Valle para:
- Consultar y gestionar informacion de prestadores.
- Administrar tecnicos y solicitantes.
- Revisar muestras e inspecciones sanitarias.
- Visualizar mapas de riesgo.
- Cargar datos masivos por CSV.
- Exportar informacion consolidada en CSV.

Alcance:
- Manual operativo para usuario final.
- Checklist basico de uso para validacion de entrega.

====================================================================
2) REQUISITOS PARA USAR EL SISTEMA
====================================================================

- Acceso web al ambiente desplegado.
- Usuario y clave vigentes.
- Navegador actualizado (Chrome, Edge, Firefox).
- Para carga masiva: archivo CSV con formato de plantilla oficial.

====================================================================
3) INICIO DE SESION
====================================================================

Ruta: /login

3.1 Ingresar
1. Abrir la pagina de login.
2. Escribir correo y contrasena.
3. Hacer clic en "Iniciar Sesion".
4. El sistema redirige al Dashboard.

3.2 Registrarse
1. Cambiar a la pestana "Registrarse".
2. Diligenciar nombre, correo y contrasena.
3. Hacer clic en "Crear Cuenta".
4. El sistema redirige al Dashboard.

3.3 Cerrar sesion
- Usar la opcion "Cerrar Sesion" en la parte inferior del menu lateral.

====================================================================
4) NAVEGACION GENERAL
====================================================================

Menu lateral principal:
- Dashboard
- Prestadores
- Tecnicos
- Solicitantes

====================================================================
5) MODULO DASHBOARD
====================================================================

Objetivo:
- Visualizar estado general del sistema.

Contenido:
- Tarjetas de conteo (prestadores, muestras, mapa de riesgo, inspecciones).
- Graficos por departamento y top municipios.
- Grafico de promedio IRCA por municipio.

Uso recomendado:
- Revisar esta pantalla al inicio de cada jornada para control rapido.

====================================================================
6) MODULO PRESTADORES
====================================================================

Ruta: /prestadores

Funciones:
- Buscar por nombre o NIT.
- Crear prestador.
- Editar prestador.
- Eliminar prestador.
- Ver detalle del prestador.

Crear prestador:
1. Clic en "Nuevo Prestador".
2. Diligenciar informacion general.
3. Diligenciar ubicacion (obligatoria).
4. Guardar.

Editar prestador:
1. Clic en icono de editar.
2. Modificar campos requeridos.
3. Guardar cambios.

Eliminar prestador:
1. Clic en icono de eliminar.
2. Confirmar accion.

Detalle de prestador:
1. Clic en icono de ver.
2. Revisar informacion consolidada en la pantalla detalle.

====================================================================
7) MODULO DETALLE DE PRESTADOR
====================================================================

Ruta: /prestadores/:id

Secciones:
- Informacion general del prestador.
- Representante.
- Ubicacion.
- Indicadores de poblacion.
- Mapa de puntos de captacion.
- Tabs con:
  - Inspecciones
  - Muestras
  - Mapa de riesgo

Notas:
- Si existen datos de bocatoma/red, aparecen tablas adicionales.

====================================================================
8) MODULO TECNICOS
====================================================================

Ruta: /tecnicos

Funciones:
- Buscar tecnicos.
- Crear, editar y eliminar tecnicos.

Campos principales:
- Nombre
- Identificacion
- Profesion
- Telefono
- Email
- Laboratorio asociado
- Ubicacion asociada

====================================================================
9) MODULO SOLICITANTES
====================================================================

Ruta: /solicitantes

Funciones:
- Buscar solicitantes.
- Crear, editar y eliminar solicitantes.

Campos principales:
- Nombre
- Estado
- Ubicacion (obligatoria)

====================================================================
10) MODULO MUESTRAS
====================================================================

Ruta: /muestras

Funciones:
- Listado paginado.
- Filtros exactos por:
  - Numero de muestra
  - NIT
  - Fecha de toma
- Detalle en modal.

Detalle de muestra:
- Informacion general.
- Resultados de laboratorio por parametro.
- Punto de muestreo.
- Prestador.
- Laboratorio.

====================================================================
11) MODULO INSPECCION SANITARIA
====================================================================

Ruta: /inspeccion

Funciones:
- Busqueda exacta por:
  - ID SIVICAP
  - Fecha de inspeccion
  - NIT del prestador
  - Nombre del prestador
- Listado en tarjetas.
- Detalle en modal.

Acceso especial:
- Boton "Completar inspecciones sin NIT" para resolver registros en staging.

====================================================================
12) MODULO RESOLVER DUPLICADOS
====================================================================

Ruta: /inspeccion/InsercionIndividual

Objetivo:
- Resolver manualmente duplicados de NIT en inspeccion_staging.

Flujo:
1. Seleccionar NIT duplicado.
2. Asignar prestador por fila.
3. Revisar contador de listas/pendientes.
4. Enviar filas listas para consolidar en inspeccion.

Funciones adicionales:
- Eliminar fila de staging.
- Cancelar cambios no enviados.

====================================================================
13) MODULO MAPA DE RIESGO
====================================================================

Ruta: /mapa

Funciones:
- Visualizacion de puntos geograficos en mapa.
- Busqueda por ID mapa, nombre/NIT de prestador, municipio.
- Panel de detalle con informacion tecnica.

Detalle tecnico disponible:
- Prestador y punto de captacion.
- Anexos y anexos2.
- Bocatoma y red.
- Resoluciones.
- Seguimiento, riesgo y seguridad.

====================================================================
14) MODULO EXPORTAR
====================================================================

Ruta: /exportar

Funciones:
- Seleccionar tipo de exportacion:
  - muestra
  - inspeccion
  - mapa_riesgo
- Cargar vista previa.
- Descargar CSV.

Recomendacion:
- Validar vista previa antes de descargar archivos finales.

====================================================================
15) MODULO SUBIR (CARGA MASIVA)
====================================================================

Ruta: /subir

Funciones:
- Elegir tipo de carga:
  - mapa de riesgo
  - muestras
  - inspecciones
- Descargar plantilla CSV oficial.
- Subir archivo y ejecutar procesamiento.

Buenas practicas:
- No modificar nombres de columnas de la plantilla.
- Cargar archivos con formato CSV.
- Revisar el mensaje final de respuesta.

====================================================================
16) GESTION DE ERRORES FRECUENTES
====================================================================

Caso: No permite ingresar
- Verificar correo y contrasena.
- Cerrar sesion y volver a intentar.

Caso: Error en carga masiva
- Verificar formato CSV y plantilla correcta.
- Revisar que haya sesion activa.

Caso: Exportacion sin datos
- Verificar tipo seleccionado.
- Validar con vista previa.

Caso: Mapa sin puntos
- Revisar existencia de coordenadas en datos fuente.
